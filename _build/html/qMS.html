qMS.addStatsDataStructToPlot(proteins, dataByProtein, ax, name, offset=0.0, markerSize=12, color='#377db8')[source]¶

addStatsDataStructToPlot adds the contents of a stats file-like datastructure to an existing plot. proteins to be plotted are listed in the non-redundent list, proteins. THESE MUST BE THE SAME PROTEINS GIVEN IN THE FIRST CALL TO PLOTSTATSDATASTRUCT

The data is in dataByProtein, the axis to add to is in ax, the name is in in name. Decent colors are red ([‘#ae2221’, ‘#d72c2b’, ‘#e78180’]) and blue ([‘#25557d’, ‘#3170a4’, ‘#5696cc’])
Parameters:
  • proteins (list) – a non-redudent list of the protein names to use (should be the IDs in dataByProtein)
  • dataByProtein (dictionary) – a dictionary (easily created by calculateStatsFile), must contain ‘vals’ for each p in proteins
  • ax (pyplot axis) – the pyplot axis to modify
  • name (string) – the name of the dataset
  • offset (float) – where to center the point (x axis), scales 0 (left edge; default) to 1.0 (right edge)
  • markerSize (int) – size of the dots (default = 12)
  • color (color) – color for the dataset (default #e31a1c)
Returns:

a pyplot axis with the data plotted

qMS.calculateStatsFile(data_dictionary, protein_set, numerator, denominator, normalization=1.0)[source]¶

calculateStatsFile takes the outputs of readCSV (data_dictionary, protein_set), a numerator, a denominator, and a normalization factor. It returns the information necessary for a stats file (vals, nvals, loc, flab)

Parameters:
  • data_dictionary – a dictionary of dictionaries, must contain uid keys for each peptide fit each element in the dict is a dict with the keys protein, ampu, ampl, amps
  • protein_set (list) – a set of protein names (must be the same names as given in the key protein in the subdict listed above)
  • numerator (list of strings) – a list of strings identifying the numerator (must be ampu, ampl, and/or amps)
  • denominator (list of strings) – a list of strings identifying the numerator (must be ampu, ampl, and/or amps)
  • normalization – a float normalization factor if you want to scale all of the values uniformly
Returns:

a dictionary of dictionaries. First key is the protein name (one of those given in protein_set). This leads to a dictionary bearing keys “vals”, “nvals”, “loc”, “flab” which are a list of values, the size of that list, the std dev of that list, the mean of that list
